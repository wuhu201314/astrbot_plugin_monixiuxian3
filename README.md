# AstrBot 插件：文字修仙

**版本:** 1.0.4
**许可证:** AGPL-3.0

---

## 简介

一款基于 AstrBot 框架的文字修仙游戏插件。提供完整的修仙成长体验，包含角色创建、双修炼路线（灵修/体修）、装备系统、突破系统、储物戒系统等核心玩法。

---

## 指令

### 基础指令
| 指令 | 说明 |
|-----|------|
| `我要修仙 [灵修/体修]` | 创建角色 |
| `我的信息` | 查看角色信息 |
| `签到` | 每日签到 |

### 修炼指令
| 指令 | 说明 |
|-----|------|
| `闭关` | 开始闭关修炼 |
| `出关` | 结束闭关修炼 |
| `突破信息` | 查看突破信息 |
| `突破 [丹药名]` | 尝试突破境界 |

### 装备指令
| 指令 | 说明 |
|-----|------|
| `我的装备` | 查看装备 |
| `装备 <物品名>` | 装备物品 |
| `卸下 <装备名>` | 卸下装备 |

### 丹药指令
| 指令 | 说明 |
|-----|------|
| `丹药背包` | 查看丹药 |
| `丹药信息 <丹药名>` | 查看丹药详情 |
| `服用丹药 <丹药名>` | 服用丹药 |

### 阁楼指令
| 指令 | 说明 |
|-----|------|
| `丹阁` | 查看丹药商店 |
| `器阁` | 查看武器商店 |
| `百宝阁` | 查看全部商品 |
| `购买 <物品名> [数量]` | 购买物品 |

### 储物戒指令
| 指令 | 说明 |
|-----|------|
| `储物戒` | 查看储物戒 |
| `存入 <物品名> [数量]` | 存入物品 |
| `取出 <物品名> [数量]` | 取出物品 |
| `丢弃 <物品名> [数量]` | 丢弃物品 |
| `赠予 <@某人/QQ号> <物品名> [数量]` | 赠予物品 |
| `接收` | 接收赠予 |
| `拒绝` | 拒绝赠予 |
| `更换储物戒 [储物戒名]` | 升级储物戒 |
